GeoTk ${geotk.version} (${release.month})

This page contains binary, javadoc and source code archive bundles of Geotoolkit.org (abridged geotk) version ${geotk.version}, released in ${release.month}.

As documented in the "Downloading GeoTk binaries" page, this download area provides a manual alternative to the automated downloading provided by the Maven repository. Maven provides fine-grain control, allowing users to fetch binary and source code on a module-by-module basis. Here, this download area provides various bundles, grouping many or all modules into a single file, as a convenience for those who do not wish to use Maven. These bundles are expandable into a single JAR file using the unpack200 tool.

 

geotk-bundle-${geotk.version}.pack.gz   (? Mb)   Most (but not all) Geotoolkit.org modules and dependencies.
geotk-bundle-storage-${geotk.version}.pack.gz   (? Mb)   Subset of geotk-bundle.pack.gz for work with JavaDB.
geotk-bundle-coverage-${geotk.version}.pack.gz   (? Mb)   Subset of geotk-bundle-storage.pack.gz for work with Grid Coverages.
geotk-bundle-referencing-${geotk.version}.pack.gz   (? Mb)   Subset of geotk-bundle-coverage.pack.gz for geo-referencing work.
geotk-${geotk.version}-javadoc.zip   (? Mb)   Aggregated javadoc of all public Geotoolkit.org classes in every module.
geotk-${geotk.version}-sources.zip   (? Mb)   The source code for all Geotoolkit.org modules.
geotk-bundle-pending-${geotk.version}.pack.gz   (? Mb)   Pending modules and their dependencies (uncommitted).
geotk-bundle-pending-${geotk.version}.zip   (? Mb)   Same than above, but as individual JAR files (uncommitted).
