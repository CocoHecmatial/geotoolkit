GradientKernelEditor demo

This demo illustrates the use of the GradientKernelEditor widget. This component allows the user to select the kernels to use for computing gradient magnitudes in a raster.

Widget javadoc: GradientKernelEditor
How to use: Modify the values in the editable fields. This demonstration widget does not use the values for performing any action.
How to make your own: Instantiate GradientKernelEditor and define a list of pre-defined kernels.
Prerequites:
  • Java 6
  • Java Advanced Imaging
  • Firefox 3 on Linux or Windows, or Internet Explorer on Windows (there is known issues with Safari, Opera and Firefox on MacOS)
