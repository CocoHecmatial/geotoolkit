CoordinateChooser demo

This demo illustrates the use of the CoordinateChooser widget. This component allows the user to specify a spatial bounding box, a temporal extent and optionally a spatial resolution.

Widget javadoc: CoordinateChooser
How to use: Modify the values in the editable fields. This demonstration widget does not use the values for performing any action.
How to make your own: Instantiate CoordinateChooser and optionally define the initial geographic and temporal coordinates.
Prerequites:
  • Java 6
  • Firefox 3 on Linux or Windows, or Internet Explorer on Windows (there is known issues with Safari, Opera and Firefox on MacOS)
