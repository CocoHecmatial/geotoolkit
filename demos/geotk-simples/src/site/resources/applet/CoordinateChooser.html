CoordinateChooser demo

This demo illustrates the use of the Geotoolkit.org CoordinateChooser widget. This component allows the user to specify a spatial bounding box, a temporal extent and optionnaly a spatial resolution.

Widget javadoc: CoordinateChooser
How to make your own: Instantiate CoordinateChooser and optionnally define the initial geographic and temporal coordinates.
Prerequites:
  • Java 6

Note that the applet is known to not work on MacOS.
