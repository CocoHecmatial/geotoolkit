GradientKernelEditor demo

This demo illustrates the use of the Geotoolkit.org GradientKernelEditor widget.

Widget javadoc: GradientKernelEditor
How to make your own: Instantiate GradientKernelEditor and define a list of pre-defined kernels.
Prerequites:
  • Java 6
  • Java Advanced Imaging

Note that the applet is known to not work on MacOS.
