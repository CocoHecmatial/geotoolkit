IIOMetadataPanel demo

This demo illustrates the use of the Geotoolkit.org IIOMetadataPanel widget. The tree shown in this applet is the structure of spatial metadata defined by Geotoolkit.org for Image I/O operations. These metadata are subsets of the ISO 19115 specification with its ISO 19115-2 extension, and are organized in a way similar to the discussion in ISO 19129. Consequently this applet can be used as a way to explore the content of the Geotk metadata format.

Widget javadoc: IIOMetadataPanel
See also SpatialMetadataFormat
How to make your own: Instantiate IIOMetadataPanel and declare the metadata to shown.
Prerequites:
  • Java 6

Note that the applet is known to not work on MacOS.
