IIOMetadataPanel demo

This demo illustrates the use of the Geotoolkit.org IIOMetadataPanel widget. The tree shown in this applet is the structure of spatial metadata defined by Geotoolkit.org for Image I/O operations. Consequently this applet provides also a way to explore the content of that metadata format.

Widget javadoc: IIOMetadataPanel
See also SpatialMetadataFormat
How to make your own: Instantiate IIOMetadataPanel and declare the metadata to shown.
Prerequites:
  • Java 6

Note that the applet is known to not work on MacOS.
