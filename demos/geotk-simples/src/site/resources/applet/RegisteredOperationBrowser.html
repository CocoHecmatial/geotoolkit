RegisteredOperationBrowser demo

This demo illustrates the use of the Geotoolkit.org RegisteredOperationBrowser widget. The tree shown in this applet is the list of registered JAI operations. Consequently this applet provides also a way to explore the JAI operations available on the host machine.

Widget javadoc: RegisteredOperationBrowser
How to make your own: Instantiate RegisteredOperationBrowser.
Prerequites:
  • Java 6
  • Java Advanced Imaging

Note that the applet is known to not work on MacOS.
