ZoomPane

Widget javadoc: ZoomPane
Purpose of this applet: Draw simple geometric shapes for demonstrating the zooms and rotations capabilities of the ZoomPane widget.
How to make your own: In order to provide useful content, developers need to create a derived class of ZoomPane and define the paintComponent(Graphics2D) method. This demo applet defines the above method for painting a triangle and a square only.
Prerequites:
  • Java 6

Note that the applet is known to not work on MacOS.
