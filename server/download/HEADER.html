Geotoolkit.org downloads

This directories contains binary, javadoc and source code of Geotoolkit.org for each release. This download area is an alternative to the Maven repository. Maven gives fine-grain control, allowing users to fetch binaries and source codes on a module-by-module basis. At the contrary this download area bundles many or all modules in a single file, as a convenience for those who don't wish to use Maven. Those two alternatives are documented on the Downloading Geotoolkit binaries page.

File Content Alternative

 geotk-bundle.pack.gz   Binaries of most (but not all) Geotoolkit.org modules as a single JAR file.   Maven repository (*.jar)
