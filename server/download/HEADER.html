Geotoolkit.org downloads

This directories contains binary, javadoc and source code archive bundles of Geotoolkit.org (abridged geotk) for each release.

As documented in the Downloading Geotk binaries page, this download area provides a manual alternative to the automated downloading provided by the Maven repository. Maven provides fine-grain control, allowing users to fetch binary and source code modules on a module-by-module basis. Here, this download area provides various bundles, grouping many or all modules into a single file, as a convenience for those who do not wish to use Maven.

The bundles are expandable into a single JAR file using the unpack200 tool.

File Content Alternative
geotk-bundle.pack.gz Most (but not all) Geotoolkit.org modules. Maven repository (*.jar)
geotk-bundle-coverage.pack.gz Subset of geotk-bundle.pack.gz for work with Grid Coverages. Maven repository (*.jar)
geotk-bundle-referencing.pack.gz Subset of geotk-bundle.pack.gz for geo-referencing work. Maven repository (*.jar)
For more information on the bundles, see the Downloading Geotoolkit.org binaries page.
geotk-javadoc.zip Aggregated javadoc of all public Geotoolkit.org classes in every module. Online javadoc
geotk-sources.zip The source code for all Geotoolkit.org modules. Maven repository (*-sources.jar)
