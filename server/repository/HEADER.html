Maven Repository: Geotoolkit.org, GeoAPI and dependencies

This repository contains Java Archive (JAR) files of Geotoolkit.org modules and its dependencies including GeoAPI. The repository follows the standard layout of Apache Maven 2, the project management tool used by Geotoolkit and other Java language projects. This repository adds to the standard public Maven 2 repository. Users are welcome to rely on this repository for their own Maven 2 builds.

This repository contains modified GeoTools binaries in which some modules have been deleted and replaced by Maven dependencies to the Geotoolkit.org equivalent modules. They are provided as a convenience for those who want functionalities not yet ported to Geotoolkit.org, but without garanteed: some tests are known to fail. Those modified GeoTools modules and JAR file names are prefixed with "gtm-", to distinguish them from the official GeoTools distribution which prefix them with "gt-" and so should not conflict in any way.

As an alternative to this Maven repository, users can download most of the Geotoolkit.org library (including its dependencies) as a single big JAR file. Those bundles are provided on the download area.
